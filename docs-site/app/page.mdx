# FlowState Docs

FlowState is packaged gateway software for e-commerce platforms that need:

- one-click crypto checkout
- shipment-linked order state transitions
- webhook-driven status sync
- XRPL-based escrow and payout workflows

## Integration Workflow

1. Install `@flowstate/gateway` in your storefront.
2. Create a project and API key (`POST /api/v1/auth/projects/create`).
3. Wrap your app with `FlowStateProvider`.
4. Add `PayButton` to product/cart surfaces.
5. Register a webhook endpoint (`POST /api/v1/webhooks/register`).
6. Process state updates from webhook events and API reads.

## API Groups

- Auth: project creation and key rotation
- Orders: checkout lifecycle and state transitions
- Shipping: rate shopping, tracking, inbound Shippo webhook
- Sellers: onboarding, orders, metrics, payouts
- Disputes: create, respond, resolve
- Platform: analytics, seller risk views, gas reports
- Webhooks: destination registration and delivery logs
- Agents: buyer/seller/admin chat routing
