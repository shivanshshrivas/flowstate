# Quickstart

Get a one-click crypto checkout button running in your storefront.

## Prerequisites

- Node.js 20+
- npm
- A wallet provider setup (wagmi + RainbowKit recommended)
- XRPL EVM Sidechain testnet access (chain ID `1449000`)
- A public HTTPS endpoint for webhooks

## 1. Install The Package

```bash
npm install @shivanshshrivas/flowstate
```

## 2. Create A Project And API Key

```bash
curl -X POST https://api.flowstate.xyz/api/v1/auth/projects/create \
  -H "content-type: application/json" \
  -d '{
    "name": "My Shop",
    "owner_email": "owner@myshop.com",
    "platform_fee_wallet": "0xYourPlatformWallet",
    "platform_fee_bps": 250,
    "contracts": {
      "escrowStateMachine": "0x33136ca845ace5a4f8CC032793AF22eC024B2ee3",
      "disputeResolver": "0x9A2C9B4CE17eE3ffD5C3Db97a04009381f457731"
    }
  }'
```

Save the returned `project_id` and `api_key` in secure server-side environment variables.

## 3. Add Provider And Button

```tsx
import { FlowStateProvider, PayButton } from "@shivanshshrivas/flowstate";

export default function ProductCheckout({ product }: { product: any }) {
  return (
    <FlowStateProvider
      config={{
        projectId: process.env.NEXT_PUBLIC_FLOWSTATE_PROJECT_ID!,
        apiKey: process.env.NEXT_PUBLIC_FLOWSTATE_API_KEY!,
        network: "testnet",
        contracts: {
          escrowStateMachine: "0x33136ca845ace5a4f8CC032793AF22eC024B2ee3",
          disputeResolver: "0x9A2C9B4CE17eE3ffD5C3Db97a04009381f457731",
          mockRLUSD: "0x257812213360Df9f3C19cAe9759B58cAcf39323d",
        },
      }}
    >
      <PayButton product={product} />
    </FlowStateProvider>
  );
}
```

<Callout type="info">
  The `PayButton` requires a connected wallet. It renders "Connect Wallet to
  Buy" when no wallet is connected and "Buy with FlowState" when connected.
  Clicking triggers a multi-step checkout overlay: shipping address, shipping
  option, price confirmation, then escrow deposit.
</Callout>

## 4. Register Your Webhook Endpoint

```bash
curl -X POST https://api.flowstate.xyz/api/v1/webhooks/register \
  -H "content-type: application/json" \
  -H "authorization: Bearer fs_live_..." \
  -d '{
    "url": "https://myshop.com/api/webhooks/flowstate",
    "events": ["order.state_changed", "dispute.created", "dispute.resolved"]
  }'
```

Persist the returned webhook `secret`, then verify `X-FlowState-Signature` in your handler.

## 5. Show Order Progress

```tsx
import { OrderTracker } from "@shivanshshrivas/flowstate";

export function OrderStatus({ order }: { order: any }) {
  return <OrderTracker order={order} />;
}
```

The `OrderTracker` displays a visual timeline showing each state in the escrow lifecycle, payout percentages released at each milestone, timestamps, and links to transactions on the XRPL EVM Explorer.

## 6. Validate End To End

1. Connect a wallet on XRPL EVM testnet (chain ID `1449000`).
2. Get test FLUSD from the faucet contract (`0x257...323d`).
3. Trigger checkout from `PayButton`.
4. Confirm order creation and shipping selection in your app.
5. Confirm you receive `order.state_changed` webhooks as Shippo tracking advances.
6. Watch partial payouts release to the seller at each milestone.
